# ♻️ Waste Watch

> **A citizen-powered waste reporting platform aligned with UN SDG 11 — Sustainable Cities and Communities.**

---

## 🌍 About the Project

**Waste Watch** is a web application that empowers citizens to take action against improper waste disposal in their communities. Built with a focus on the United Nations Sustainable Development Goal **SDG 11 – Sustainable Cities and Communities**, the platform bridges the gap between residents and municipal authorities by making waste reporting fast, simple, and accessible.

Whether it's an overflowing bin on your street, illegal dumping near a park, or uncollected garbage — Waste Watch gives you a voice.

---

## ✨ Features

| Feature | Description |
|---|---|
| 🚨 **Report Waste Issues** | Submit waste or garbage complaints directly from your browser |
| ♻️ **Recycling Tips** | Access practical tips on how to recycle and reduce waste at home |
| 📚 **Waste Categories Guide** | Learn the difference between biodegradable, recyclable, e-waste, and more |

---

## 🎯 SDG Alignment

This project directly contributes to:

- **SDG 11 – Sustainable Cities and Communities**
  > *"Make cities and human settlements inclusive, safe, resilient and sustainable."*
  
  Specifically targets **SDG 11.6** — reducing the adverse environmental impact of cities, including by paying special attention to air quality and municipal waste management.

## 🚀 Getting Started

No installation or build step needed. Just clone and open.

```bash
# 1. Clone the repository
git clone https://github.com/mayurkadam2/Waste-Watch-SDG-Project.git

# 2. Navigate into the project folder
cd Waste-Watch-SDG-Project

# 3. Open index.html in your browser
open index.html
```

Or simply double-click `index.html` in your file explorer.
